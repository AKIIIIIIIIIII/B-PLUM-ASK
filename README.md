## Overview

`Plum.B` 是一个以简洁排版和东方气质为核心的 SwiftUI 小应用。  
用户输入 3 个数字后起卦，应用会展示卦象、卦辞、象传、动爻与六爻结果。

## Features

- SwiftUI 构建的极简输入页与结果页
- 基于 3 个输入数字生成卦象结果
- 展示卦辞、象传、动爻与六爻内容
- 统一的中英文字体与留白节奏设计

## Tech Stack

- Swift
- SwiftUI
- Xcode

## Run Locally

1. 使用 Xcode 打开 `Plum.B.xcodeproj`
2. 选择模拟器或真机
3. 运行 `Plum.B`
